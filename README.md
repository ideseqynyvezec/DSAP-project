# 虛擬貨幣與外匯「三角套利」機會尋找器

## Proposal Report

### 動機與目標
我以前就常常聽說過外匯與虛擬貨幣市場中的「三角套利」策略，覺得這種利用市場定價無效率來獲利的機制非常神奇，一直想要親自探索它的真實性與運作邏輯。藉由這次專題的機會，我希望能將課堂上學到的「資料結構」與「進階程式設計」理論轉化為實際的工具。本專案的目標是開發一款輕量級的套利路徑尋找器。系統將讀取多種貨幣間的匯率，透過演算法快速偵測是否存在套利空間，並輸出具體的交易路徑與預期報酬率。

### 預期功能
本專案將專注於最核心的邏輯實作：
1. 讀取基礎匯率：讀取一份包含 4~5 種貨幣匯率的本地靜態 CSV 或 JSON 檔案。
2. 建構基礎圖形：將讀取的資料轉換為基本的有向圖（Directed Graph）結構。
3. 基礎三角套利偵測：實作基礎的搜尋邏輯（深度優先搜尋 DFS），嚴格限制只尋找「3個幣種」的套利循環（例如：TWD -> USD -> JPY -> TWD）。
4. 文字結果輸出：在終端機單純印出找到的路線以及相乘後的匯率數值。

### 使用技術
* 程式語言：Python / C++
* 資料結構實作：使用基本的 Hash Map 或 Adjacency List 來實作 Graph。
* 開發工具：Git / GitHub 進行版本控制。

### 時程規劃
* Week 7-8：繳交 Proposal，建立 GitHub Repo，並準備一份只有 5 種貨幣的假資料。
* Week 9-10：將假資料讀入程式，實作 Graph 資料結構，並寫出尋找長度為 3 的 DFS 搜尋邏輯。
* Week 11：程式能成功讀取資料並跑出結果（即使介面很簡陋也沒關係），發布 Prototype Release。
* Week 12-13：不增加新功能。專注於修復程式 Bug，處理極端狀況（例如找不到套利路徑時程式不會崩潰），並補齊程式碼註解。
* Week 14：撰寫 README.md 的最終報告，並開始錄製 5 分鐘 Demo 影片。
* Week 15：確認所有文件與影片連結無誤，發布 Final Report Release。

## Prototype Report

### 目前進度
截至 Week 11，本專案已順利從「靜態假資料」跨越至「真實大數據回測」，具體完成事項如下：
1. 自動化資料獲取： 撰寫 Python 腳本（搭配 yfinance 套件）自動抓取 Yahoo Finance 的真實法定貨幣（如 USD, EUR, JPY, TWD）歷史報價，並清洗、輸出為標準化的 CSV 時間序列格式。
2. 核心回測引擎建置： 於 C++ 中實作 HistoricalBacktester 類別，將龐大的歷史資料成功映射為時間序列資料結構，內部並以相鄰串列儲存每個時間點的圖結構。
3. DFS 演算法實作與效能量測： 成功實作 DFS，並導入 <chrono> 函式庫進行高精度計時。目前引擎已能以約 60 毫秒的極佳效能，掃描超過 500 個小時的歷史快照，並精確抓出匯率相乘大於 1 的三角路徑。

### 遇到的困難
1. 環境編碼與字串解析： 在 Windows 環境下輸出 C++ 終端機中文時遇到 Big5/UTF-8 的亂碼衝突，已透過呼叫 windows.h API 強制切換編碼解決。
2. 免費的 API 很難抓到大量、高精度的分鐘級法定貨幣歷史資料。
3. 量化數據的「完美幻覺」： 在初步回測中，程式於 519 個快照內找到了高達 516 次的套利機會。這可能是因為免費的歷史資料僅提供中間價，忽略了真實市場的買賣價差與手續費摩擦成本；且以「小時」為單位的取樣頻率，容易產生不同貨幣對報價時間未完全對齊的合成誤差。

### 下一步計畫
1. 提升數據取樣頻率與跨足虛擬貨幣市場： 由於 1 小時一次的法幣快照間隔過大，無法捕捉真實的瞬時套利空間。接下來將撰寫串接 Binance API 的腳本，獲取精確至「分鐘級」的真實虛擬貨幣歷史資料 (如 BTC, ETH, SOL 等)，藉由高波動性與高頻率的真實數據來對演算法進行壓力測試。
2. 導入交易摩擦成本： 修改目前的 DFS 演算法判斷邏輯，加入真實交易所的買賣價差與手續費門檻，剔除因數據不完美而產生的虛假套利，過濾出真正具備實務可行性的獲利機會。
3. 實作核心圖論演算法 (Bellman-Ford)： 把匯率轉換為負對數 $(-\log R)$，利用 Bellman-Ford 演算法尋找圖中的負權環，發掘潛在的長路徑多角套利空間。
4. 演算法效能比較與總結： 針對 DFS 與 Bellman-Ford 兩者在處理相同巨量數據時的「執行時間」與「記憶體消耗」進行實際量化比較，將結果彙整至 Final Report，並錄製最終的 Demo 影片。
